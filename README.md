# SkillTap
You have to react fast and destroy the falling beam with the TNTs.
How fast can you react and tap.
You have to press the right button when the beams are overlapping with the TNTs
## Controls
|TNTs:|1|2|3|4|
|-:|:-:|:-:|:-:|:-:|
BUTTONs:|LEFT|BOTTOM|A|B|

Green Light = Good\
Red Light = Missed
